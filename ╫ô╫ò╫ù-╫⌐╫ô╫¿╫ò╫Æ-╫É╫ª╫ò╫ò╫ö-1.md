## 8) חיבור דומיין מותאם (כשתרצה)
1. קנה דומיין (Namecheap/Cloudflare, ~50 ₪/שנה).
2. ב-Netlify: Domain settings → Add a custom domain → הזן את הדומיין.
3. Netlify ייתן לך רשומות DNS (או name servers) — אתה מזין אותם אצל רשם הדומיין. **אני לא נוגע ב-DNS או בהרשמה בשמך** — זה מידע שאתה מזין בעצמך; אני אכוון צעד-צעד.
