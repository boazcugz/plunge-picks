## 1. חזון

**Plunge Picks** — פלטפורמת החלטה (decision-support) בנישת טבילות קרח / ice baths / chillers / ציוד התאוששות ביתי. המטרה: לעזור לקונה לבחור מוצר יקר בצורה כנה ומבוססת-מחקר, ולהרוויח דרך שיווק שותפים (affiliate). **לא** בלוג גנרי, ולא אתר שממציא ביקורות.

עקרון-על: **קודם בסיס אמין וממיר, אחר כך הרחבה.** לא מרחיבים לעשרות עמודים לפני שהבסיס הקיים אמין, עובד וממיר.

## 3. ארכיטקטורה

- אתר **סטטי**: HTML + CSS + vanilla JS. **ללא** build system, framework או backend.
- עיצוב: קובץ יחיד `style.css` עם CSS variables (design tokens). פלטה: כחול כהה `#0d1b2a`, ציאן `#14b8c4`, זהב `#f4a63b`.
- נתונים ריכוזיים:
  - `products.js` → `window.PLUNGE_PRODUCTS` (מסד המוצרים)
  - `affiliate-programs.js` → `window.PLUNGE_AFFILIATE_PROGRAMS` (סטטוס תוכניות שותפים)
  - `analytics.js` → `window.plungeTrack` (מעקב אירועים ללא PII)
- 4 כלים אינטראקטיביים (vanilla JS): Product Finder, Cost Calculator, Comparison Builder, Ice-vs-Chiller.

## 4. כללי עיצוב קבועים

- Header אחיד (`nav2`) ו-Footer אחיד (`footer2`) בכל העמודים.
- מובייל-first: תפריט המבורגר מתחת ל-760px, ללא גלישה אופקית.
- אין לשבור את מערכת ה-design tokens — צבעים/מרווחים דרך המשתנים ב-`:root`.
- Breadcrumbs ויזואליים בעמודי תוכן וכלים.

## 5. כללי אמינות (Non-negotiables)

1. אין להמציא נתוני מוצר, בדיקות פיזיות, ניסיון אישי, מדידות, רעש, צריכת חשמל.
2. נתון לא-מאומת מסומן `verify:true` / "Not confirmed" ואינו מוצג כעובדה.
3. אין ביקורות/דירוגים/מומחים/מבקר רפואי/תגי אמון מזויפים.
4. מפרידים בין **טענת יצרן** לבין **מסקנת האתר**.
5. טענות כמו "שקט", "עמיד", "מתאים לדירה" — דורשות מקור או ניסוח מסויג.
6. תמונת AI שאינה תמונת מוצר אמיתית מסומנת "Illustrative".
7. אין טענות רפואיות; קיים Medical Disclaimer עם תנאי עצירה.

## 6. כללי שותפים ומחירים

- המשפט של אמזון בנוסח מדויק מילה במילה: **"As an Amazon Associate I earn from qualifying purchases."**
- גילוי שותפים בסמוך לכל המלצה מסחרית.
- **אין מחיר אמזון סטטי** ללא מקור מאושר → משתמשים בקטגוריות (Budget / Mid-range / Premium) + "Check current price".
- לפני Product Advertising API (דורש 3 מכירות) — אין מחירים/תמונות ממשקי אמזון.
- כל קישור שותף: `rel="sponsored nofollow noopener"`, `target="_blank"`.
- אין להציג מותג כשותף פעיל לפני אישור התוכנית.
- קישורי חיפוש גנריים מוחלפים ב-Special Links מדויקים.
